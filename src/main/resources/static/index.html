API Documentation

Endpoint: /api/encode

Encode a message into an image

        POST /api/encode
        Content-Type: multipart/form-data

        {
            "image": (file),
            "message": "Your secret message here"
        }

Endpoint: /api/decode

Decode a message from an image

        POST /api/decode
        Content-Type: multipart/form-data

        {
            "image": (file)
        }
